在线AI

图片风格迁移

  • Ostagram
    Intro: 转换效果极佳，有较多预设风格，可选择免费的低清模式或者付费的高清模式，免费模式需要花费较长时间转化。

  • Arbitrary Style Transfer in the Browser
    Intro: 转换效果一般，除单风格迁移还提供，两图混合风格迁移。转换速度快，可选择神经网络模型。该项目开源，主页提供github地址。

图像生成

  • MakeGirlsMoe
    Intro: 随机生成二次元头像，可设置性别、发色、瞳色、发型等参数。

  • Crypko
    Intro: MakeGirlsMoe作者的另一项目，看起来功能强大，其测试版和社区暂时关闭。

资源站

图片

  • Unsplash
    Intro: 提供大量各种分类的超清图片，提供可获取固定大小高清图的API (本博客头图即使用该API)

  • Nipic图片下载
    Intro: 非图片网站会员下载图片，购买账号（页面提供购买接口），下载消耗账号中的点数。

百度文库下载

  • 文档服务
    Intro: 非百度会员下载百度文库文档，购买账号（页面提供购买接口），下载消耗账号中的点数。

Github单文件(夹)下载

  • DownGit
    Intro: 提供Github单文件(夹)下载，输入文件对应url即可下载。完全免费。

博客框架

  • Hexo
    Intro: 提供开源博客框架，本网站即使用该框架。

在线设计

  • 稿定设计
    Intro: 在线设计网站，包括海报、logo、ppt、h5、简历、网店设计等。

  • 墨刀
    Intro: 小程序UI在线设计，提供多种预设组件，支持点击跳转页面。

Geek

服务器

  • Vultr
    Intro: 国外服务器购买，首次注册赠送50刀券（注意在有效期使用完，过期则自动失效，即使正在使用）。按小时收费（先使用优惠卷金额），可自由删除新建服务器，多国多地服务器可选。很多被墙，注册后最好先尝试是否能PING通（见PING检测）。

PING检测

  • 多个地点Ping服务器
    Intro: 输入IP即可查询各地ping，可用于检测国外服务器是否被墙。

游戏

LoL

  • LoL Dodge Game
    Intro: LoL基本功（走位、走a、非指向技能施放等）在线训练。

  • OPGG
    Intro: LoL大数据中心，可查韩服战绩，职业选手OB等。
